	Voici le jeu jeu DefendTheBackrooms qui comme son nom l'indique un tower defense dans les backrooms,(c'est faux, c'est une map sur la terre ferme où des tourelles tuent des pauvres petits crabes)

------Cette version est encore une beta, la réelle map, réaliste, générée procéduralement n'est pas encore présente.



REQUIS:
Pour utiliser le jeu assurez vous d'avoir la bibliothèque SDL2, lsdl2-ttf, lsdl2-mixer, lsdl2-image





COMPILEZ avec la ligne suivante : 

gcc -o testsdl testsdl.c fonctions.c -lSDL2 -lSDL2_image -lSDL2_ttf -lSDL2_mixer -lm


(J'expliquerai ultèrieurement comment se servir des fonctions du header fonctions.h, écrites dans le code fonctions.c)
